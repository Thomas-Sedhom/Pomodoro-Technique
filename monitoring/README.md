## Prerequisites

- Docker
- Docker Compose

## Setup

### Clone the Repository

Clone the repository containing the `docker-compose.yaml` file.

### Run Docker Compose

Start the services using Docker Compose:

```sh
docker compose up
```

**Stopping the Setup**
To stop the Docker Compose setup, run:
```bash
docker compose down
```
